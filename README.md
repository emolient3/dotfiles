Mis configuraciones personales para un entorno **Hyprland** minimalista en Arch Linux.

## 📦 Stack

### 🪟 Window Manager
| Paquete | Descripción |
|---|---|
| `hyprland` | Compositor Wayland dinámico |
| `waybar` | Barra de estado personalizable |

### 💻 Terminal
| Paquete | Descripción |
|---|---|
| `kitty`  | Emulador de terminal|
| `zsh` | Shell interactiva |

### 📁 File Manager
| Paquete | Descripción |
|---|---|
| `lf` | Gestor de archivos en terminal |
| [`ctpv-git`](https://github.com/NikitaIvanovV/ctpv) | Previsualizador de archivos para `lf` |
### 🛠️ AUR
| Paquete | Descripción |
|---|---|
| `bat` | resaltado de sintaxis |
| `chafa` | Renderizado de imágenes en terminal |
| `ffmpeg` | Procesamiento de audio/video |

### ✏️ Editor
| Paquete | Descripción |
|---|---|
| `neovim` | Editor de texto modal |

### 🔤 Fuentes
| Paquete | Descripción |
|---|---|
| `ttf-jetbrains-mono-nerd` | Fuente monoespaciada con iconos Nerd Font |
