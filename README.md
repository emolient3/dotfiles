## 📦 Stack

| Paquete | Descripción |
|---|---|
| [`hyprland`](https://github.com/hyprwm/hyprland) | Compositor Wayland dinámico |

### 💻 Terminal
| Paquete | Descripción |
|---|---|
| [`kitty`](https://github.com/kovidgoyal/kitty)  | Emulador de terminal|
| `zsh` | Shell interactiva |

| Paquete | Descripción |
|---|---|
| `lf` | Gestor de archivos en terminal |
| [`ctpv-git`](https://github.com/NikitaIvanovV/ctpv) | Previsualizador de archivos para `lf` |
